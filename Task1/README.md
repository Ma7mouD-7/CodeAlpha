## About The Project

Shelfy is Library Application to find, explore, and orgnaize your Bookshelf
offers Books, Authors, and Categories
with User-friendly GUI 

## Built With

* Kotlin
* Openlibrary API
* XML
* MVVM
* Retrofit
* Firebase
